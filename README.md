## Описание

Express-приложение для расчёта Индекса Массы Тела (ИМТ).

## Функционал

- Расчёт ИМТ по весу и росту
- Категоризация результатов
- CRUD операции для записей
- Валидация входных данных
- Логирование запросов
- Веб-интерфейс
